# 2025/2/9 学習日記  
### Ninja Code 初級1~3  
- 初級1.  
    - const(定数)は宣言と代入同時。  
    - let(変数)は同じ変数名を再度宣言NG  
(ex:let number = 8  
&emsp;   let number = 26 NG)
    - constの値は変更不可
- 初級2.
    - 変数名同士の計算可
- 初級3.
    - 正規表現による文字列の検索・置換
        - let text = テキスト(変数名).replece("A","B")  
テキスト(変数)から"A"に該当するものを"B"に置き換えるメソッドの1つである
また、Bをブランクにすることで、Aを無に帰す(取り除く)ことが可能
